# Note for Create React App Tool

## Installation

`[sudo] npm i -g create-react-app`

## Create App

`create-react-app app-name`

## Run App

```
cd app-name
npm start
```

## More information

```
cd app-name
vi README.md
```

## Uninstallation

It's no longer use create-react-app we use npx instead of

`sudo npm uninstall -g create-react-app`
